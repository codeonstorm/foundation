| Short Form  | Full Form                           | Simple Meaning                                                  |
| ----------- | ----------------------------------- | --------------------------------------------------------------- |
| **S — SRP** | **Single Responsibility Principle** | One class should have one responsibility / one reason to change |
| **O — OCP** | **Open/Closed Principle**           | Open for extension, closed for modification                     |
| **L — LSP** | **Liskov Substitution Principle**   | Child class should safely replace parent class                  |
| **I — ISP** | **Interface Segregation Principle** | Create small focused interfaces                                 |
| **D — DIP** | **Dependency Inversion Principle**  | Depend on abstractions, not concrete implementations            |
