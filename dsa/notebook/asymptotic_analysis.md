**Asymptotic analysis** is the study of how an algorithm's **time complexity** and **space complexity** **grow** as the input size (`n`) becomes very large. It focuses on the **growth rate** of the resources (time and/or space) required by the algorithm rather than the exact execution time or memory usage.

### In simple words

* **Time Complexity** → How the **execution time grows** as `n` increases.
* **Space Complexity** → How the **memory usage grows** as `n` increases.

### Memory Rule

> **Asymptotic Analysis = Growth Rate Analysis**

It answers the questions:

* How does the **time** grow?
* How does the **space** grow?

when the input size becomes very large (`n → ∞`).



###
asymptotic analysis shows the growth rate of time and/or space with respect to the input size, not the exact time or exact memory used.
